Ember

Introduction

EmberJS was created by Yehuda Katz, a member of the Rust Core Team, however, he lists himself as co-creator of Ember on his own site (Katz, 2015). The framework uses Handlebars as it's templating engine, which Katz also created (Katz, 2015).

How it works

The entry point for this is index.html, however, you can't see where the base template is injected. The body recieves a class of ember-application by default, but it is added to it somewhere that I couldn't find. application.hbs is the base template, yet there is no reference of where it will be inserted.
